Bubble Sort

Bublesort is simplest ordering method like bubble that float to the surface. Bubble sort will compare two numbers nearly if it found that some number not ordered bubble sort will change position of that number and ordering them. The process will continued until all numbers have been ordering.

Ordering method steps

from Least to Greatest

  • 1. Compare numbers at index1 and 2 if index2 has value less than index1 change the index.
  • 2. Compare index1 with index2 continued to the last index check if there are any numbers that has less value than index before . We call this work one round of process.
